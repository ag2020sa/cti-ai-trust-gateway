# Synthetic benchmark

All reports are original synthetic text created for this repository and licensed under
Apache-2.0. Run `python scripts/build_synthetic_benchmark.py` to reproduce ten bilingual base
reports, gold bundles, and 100 deterministic mutations using seed `20260717`.

The mutation manifest records provenance, language, expected verdict, finding category, and
license. It is a regression corpus, not a claim about real incidents or infrastructure.

CTIBench is not vendored because its CC BY-NC-SA terms are unsuitable for the core Apache-2.0
repository. Users may download and evaluate it separately under its own license.
